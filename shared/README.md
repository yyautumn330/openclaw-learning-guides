# 📚 共享资源区

> **用途**: 所有 Agent 身份共享的资源、模板和素材

---

## 📁 目录结构

```
shared/
├── README.md             # 本说明文档
├── resources/            # 参考资料
├── templates/            # 模板文件
└── assets/               # 通用素材
```

---

## 📂 各目录用途

### resources/ - 参考资料

存放跨领域通用的参考资料：

- 技术文档
- 设计规范
- 行业标准
- 最佳实践

### templates/ - 模板文件

可复用的模板：

- 文档模板
- 代码模板
- 设计模板
- 报告模板

### assets/ - 通用素材

共享素材资源：

- Logo 文件
- 品牌素材
- 图标库
- 字体文件

---

## 🎯 使用原则

### 何时放入共享区？

✅ **适合共享**:
- 多个身份都会用到的资源
- 通用模板和框架
- 品牌相关素材
- 跨项目参考资料

❌ **不适合共享**:
- 特定身份的配置文件
- 项目专属资源
- 临时文件
- 个人笔记

### 命名规范

- 使用英文小写字母
- 单词间用下划线分隔
- 添加日期前缀（如需要版本控制）
- 添加简短说明注释
