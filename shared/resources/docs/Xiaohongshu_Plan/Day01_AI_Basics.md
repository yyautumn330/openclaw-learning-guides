### 正文内容

```
👋 大家好，我是 CodeMaster，一个爱说人话的程序员～

从今天开始，我会用 30 天时间，带你从零开始学 AI！
每天一篇，通俗易懂，保证你能看懂！

今天是第一天，先搞懂最常见的 3 个概念👇

━━━━━━━━━━━━━━━━━━━━

🤔 AI / 机器学习 / 深度学习 到底是啥？

很多人被这些术语搞晕了，其实它们的关系很简单！

用「做菜」来比喻你就懂了👇

━━━━━━━━━━━━━━━━━━━━

1️⃣ AI（人工智能）= 整个厨房

就是让机器像人一样「智能」地做事
比如：下棋、开车、聊天、画画...

📌 范围最大，是个大概念

━━━━━━━━━━━━━━━━━━━━

2️⃣ 机器学习 = 厨师的学习方法

传统编程：人告诉机器每一步怎么做
机器学习：机器自己从数据中学习规律

就像：
❌ 传统：菜谱写死每一步
✅ 机器学习：厨师尝菜后自己改进

📌 是 AI 的一种实现方式

━━━━━━━━━━━━━━━━━━━━

3️⃣ 深度学习 = 更厉害的学习方法

用「神经网络」模拟人脑
数据越多，学得越好

就像：
✅ 深度学习：厨师有 10 年经验，越做越好吃

📌 是机器学习的一种，效果最好

━━━━━━━━━━━━━━━━━━━━

📊 一张图看懂关系：

┌──────────────────┐
│      AI          │  ← 最大概念
│  ┌────────────┐  │
│ │  机器学习   │  │  ← AI 的子集
│ │ ┌────────┐ │  │
│ │ │深度学习│ │  │  ← 机器学习的子集
│ │ └────────┘ │  │
│  └────────────┘  │
└──────────────────┘

━━━━━━━━━━━━━━━━━━━━

💡 举个例子：识别猫的照片

🔹 AI：让机器识别图片内容
🔹 机器学习：给机器看很多猫的照片，让它自己学
🔹 深度学习：用神经网络，识别更准确

━━━━━━━━━━━━━━━━━━━━

✅ 总结一下：

• AI 是大概念，目标是让机器智能
• 机器学习是 AI 的一种方法，让机器从数据学习
• 深度学习是机器学习的一种，用神经网络，效果最好

它们是从属关系，不是并列关系哦！
```
