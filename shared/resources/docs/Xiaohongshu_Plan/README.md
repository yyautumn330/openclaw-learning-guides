## 💡 内容技巧

### 标题公式

**有效标题 = 数字 + 痛点 + 解决方案**

示例：
- ✅ 《0 基础！3 分钟搞懂 AI》
- ✅ 《5 个公式，让 AI 听你话》
- ✅ 《打工人必备！10 个 AI 工具》

### 正文结构

```
1. 引入（场景/痛点）
2. 核心内容（分点）
3. 示例/演示
4. 总结（要点）
5. 互动（提问）
```

### 提高互动

**结尾话术：**

📅 明天讲《XXX》，记得来看！

## 📈 优化策略

### 每周复盘

**问题：**
1. 哪篇阅读最高？为什么？
2. 哪篇互动最好？为什么？
3. 用户最关心什么？
4. 下周如何改进？

**行动：**
1. 复制成功经验
2. 改进不足之处
3. 调整内容方向
4. 优化发布时间

### 内容迭代

根据数据调整：
- 阅读量低 → 优化标题/封面
- 点赞少 → 提升内容质量
- 收藏少 → 增加实用价值
- 评论少 → 加强互动引导
