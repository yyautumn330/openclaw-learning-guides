
- 好提示词 vs 坏提示词

### 5 个公式
```
1. 角色 + 任务
   "你是一位老师，请解释..."

2. 背景 + 要求
   "我正在学习 AI，请用简单语言..."

3. 示例 + 模仿
   "像这样：XXX，请模仿..."

4. 分步 + 详细
   "第一步...第二步...第三步..."

5. 限制 + 输出
   "用 100 字以内，列出 3 个要点..."
```
