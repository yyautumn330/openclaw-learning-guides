## ⚠️ 注意事项

### 需要额外配置的技能
- **gog** - 需要 Google OAuth 认证
- **notion** - 需要 Notion API Token
- **api-gateway** - 需要各平台 OAuth
- **nano-banana-pro** - 需要 Gemini API key

### 功能重叠
- `humanizer` 和 `humanize-ai-text` 功能相似，选一个即可
- `tavily-search` 和 `brave-search` 都是搜索，推荐 brave-search (更成熟)

### 版本选择
- 优先选择版本数多的技能 (表示持续维护)
- 查看最近更新时间和作者活跃度
