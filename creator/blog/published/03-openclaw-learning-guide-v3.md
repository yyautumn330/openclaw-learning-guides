## 🐛 常见问题

### Q1: 技能安装失败？

**原因**: 网络问题或技能源不可用

**解决**:
```bash
# 检查网络连接
ping clawhub.com

# 清除缓存重试
openclaw skills clear-cache
openclaw skills install <skill-name>

# 手动下载技能包
# 从 clawhub.com 下载 .zip 后本地安装
```

---

### Q2: 编译错误如何处理？

**步骤**:
1. 仔细阅读错误信息
2. 搜索错误代码
3. 检查 API 版本兼容性
4. 简化实现或使用替代方案

**案例**: HarmonyOS API 不兼容
```typescript
// 错误：API 不存在
await geolocationManager.subscribeLocation()

// 解决：使用替代 API
await geolocationManager.requestLocationUpdates()
```

---

### Q3: 备份文件过大？

**原因**: 包含了模型文件、虚拟环境等

**解决**: 创建 `.gitignore`
```gitignore
# AI 模型
*.safetensors
*.ckpt
*.pth

# 虚拟环境
venv/
.env/

# 依赖包
node_modules/

# 构建输出
build/
dist/
```

**优化效果**: 6.7GB → 140MB (减少 98%)
