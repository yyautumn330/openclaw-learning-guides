🚀 快速开始（5 分钟搞定）

⏰ 步骤 1：安装 OpenClaw（2 分钟）

打开终端，执行：

npm install -g openclaw

验证安装：

openclaw --version

看到版本号就说明安装成功了！🎉

⏰ 步骤 2：配置 AI 模型（2 分钟）

OpenClaw 支持多种 AI 模型：

【免费模型】新手推荐👇
• Qwen（通义千问）阿里出品
• GLM（智谱 AI）清华系
• Kimi 月之暗面

【付费模型】效果更好👇
• GPT-4 OpenAI
• Claude Anthropic
• Gemini Google

配置命令：

openclaw configure

然后选择模型提供商，输入 API Key
（去官网申请，免费额度够用）

API Key 获取地址👇
• Qwen：dashscope.console.aliyun.com
• GLM：open.bigmodel.cn
• GPT-4：platform.openai.com

⏰ 步骤 3：启动助手（1 分钟）

openclaw gateway

看到🦞 OpenClaw 标志就说明启动成功了！

启动后，你就可以在终端和 AI 对话了！
