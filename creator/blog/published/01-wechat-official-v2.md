### 01 为什么选择 OpenClaw？

先说说我为什么折腾这个。

之前一直用 ChatGPT Plus，每月$20，一年下来也是笔不小的开支。

关键是：
- 每次新对话都要重新介绍自己
- 数据存在人家服务器上，心里不踏实
- 只能被动回答问题，不会主动干活

OpenClaw 完美解决了这些问题：

**1. 数据本地存储**

所有配置、记忆、聊天记录都存在你自己电脑上，不上传云端。

隐私安全，完全掌控。

**2. 长期记忆**

AI 会记住你的：
- 名字、时区、工作时间
- 技术栈偏好
- 正在进行的项目
- 重要决定和习惯

比如我说"帮我看看那个太阳追踪项目"，AI 知道我说的是 SunTracker（我的 iOS 应用），直接打开对应目录。

**3. 主动模式**

传统 AI 只能被动响应，OpenClaw 可以主动执行任务：

每天早上 8 点自动推送：
```

🌤️ 天气：晴，15-22°C

```

开会前 15 分钟自动提醒，项目进度主动汇报。

### 02 5 分钟快速部署

**步骤 1: 安装 Node.js** (2 分钟)

macOS 用户：
```bash
brew install node
node --version
```

Windows 用户：
1. 访问 https://nodejs.org/
2. 下载 LTS 版本
3. 双击安装

**步骤 2: 安装 OpenClaw** (1 分钟)
```bash
npm install -g openclaw
openclaw --version
```

**步骤 3: 配置 AI 模型** (2 分钟)

推荐新手用阿里通义千问（Qwen）：
1. 访问 https://dashscope.console.aliyun.com/
2. 注册/登录
3. 创建 API Key（100 万 tokens 免费额度）
4. 配置：
```bash
openclaw configure
# 按提示输入 API Key
```

**步骤 4: 启动！** (30 秒)
```bash
openclaw gateway start
```

看到🦞标志就成功了！

### 05 安全吗？

完全不用担心。

OpenClaw 的安全机制：
- ✅ 数据本地存储
- ✅ API Key 加密保存
- ✅ 技能沙箱隔离
- ✅ 敏感操作需确认
- ✅ 支持完全离线

**OpenClaw 不会**：
- ❌ 收集你的聊天记录
- ❌ 上传你的文件到第三方
- ❌ 用你的数据训练模型
- ❌ 向广告商出售数据

### 07 常见问题

**Q: 需要编程基础吗？**

A: 基础使用不需要！复制粘贴命令就行。想开发技能才需要编程。

**Q: Windows 能用吗？**

A: 可以！推荐装 WSL2，然后按 Linux 流程操作。

**Q: 微信会封号吗？**

A: 有风险，用小号测试。推荐用飞书或 Telegram。

**Q: 数据丢了怎么办？**

A: 定期备份！
```bash
cp -r ~/.openclaw/workspace ~/backup/
```
