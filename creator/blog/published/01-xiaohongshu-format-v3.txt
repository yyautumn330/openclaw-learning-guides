### 🚀 5 分钟快速部署

**步骤 1: 安装 Node.js** (2 分钟)
```bash
# macOS
brew install node

# 验证
node --version
```

**步骤 2: 安装 OpenClaw** (1 分钟)
```bash
npm install -g openclaw
openclaw --version
```

**步骤 3: 配置 AI 模型** (2 分钟)
- 去阿里通义千问申请免费 API Key
- 100 万 tokens 免费额度！
- 配置命令：
```bash
openclaw configure
# 按提示输入 API Key
```

**步骤 4: 启动！** (30 秒)
```bash
openclaw gateway start
```

看到🦞标志就成功了！

### ❓ 常见问题

**Q: 需要编程基础吗？**
A: 基础使用不需要！复制粘贴命令就行。想开发技能才需要编程。

**Q: Windows 能用吗？**
A: 可以！推荐装 WSL2，然后按 Linux 流程操作。

**Q: 微信会封号吗？**
A: 有风险，用小号测试。推荐用飞书或 Telegram。

**Q: 数据丢了怎么办？**
A: 定期备份！
```bash
cp -r ~/.openclaw/workspace ~/backup/
```
