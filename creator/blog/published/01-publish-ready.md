## 🎨 配图方案

### 方案 A: 快速发布 (推荐新手)

**用现有素材** - 15 分钟搞定

1. **封面图** - 用 Keynote/Canva 制作
   - 蓝色渐变背景
   - 标题："5 分钟搭建 AI 助手"
   - 尺寸：1242×1660 px

2. **终端截图** - 实际执行命令截图
   - 安装过程
   - 版本验证
   - 启动服务

3. **信息图** - 用 Keynote 制作
   - 功能介绍
   - 命令速查

**优点**: 快速、真实、可信  
**缺点**: 视觉冲击力一般

---

### 方案 B: AI 生成 (推荐)

**用 nano-banana-pro** - 需要 API Key

1. **配置 API**
   ```bash
   openclaw config set tools.nano-banana-pro.api_key "你的 Gemini API Key"
   ```

2. **生成封面**
   ```
   科技感封面图，蓝色渐变背景，AI 助手概念，机器人图标，
   简洁现代，标题文字"5 分钟搭建 AI 助手"，发光效果，
   未来感，高分辨率，3:4 比例
   ```

3. **生成内页**
   - 功能介绍图
   - 技能列表图
   - 命令速查图

**优点**: 视觉冲击强、专业、统一风格  
**缺点**: 需要 API Key、需要调试提示词

**获取 API**: https://aistudio.google.com/apikey

---

### 方案 C: 混合方案 (平衡)

- P1 封面：AI 生成 (吸引点击)
- P2-P9：终端截图 + 简单信息图

**优点**: 平衡速度和质量  
**缺点**: 风格可能不统一
