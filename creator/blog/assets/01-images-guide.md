## 📊 图片优化建议

### 小红书图片要求

- **格式**: JPG/PNG
- **大小**: <10MB/张
- **数量**: 4-9 张
- **比例**: 
  - 封面 3:4 (最佳)
  - 内页 1:1 或 16:9

### 优化命令

```bash
# 压缩图片
ffmpeg -i input.png -q:v 8 output.jpg

# 调整尺寸
ffmpeg -i input.png -vf "scale=1242:1660" output.png
```
