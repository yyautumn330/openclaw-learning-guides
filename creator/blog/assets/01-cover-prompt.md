## 🛠️ 生成命令

```bash
# 使用 nano-banana-pro 生成
openclaw "生成封面图：科技感蓝色主题，AI 助手，5 分钟搭建，3:4 比例"

# 或使用 openai-image-gen
openclaw "生成 DALL-E 3 图片：科技感封面，蓝色主题，AI 助手概念"
```
