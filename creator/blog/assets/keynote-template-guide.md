## 📐 模板规格

### 小红书图片尺寸

| 类型 | 比例 | 推荐尺寸 | 用途 |
|------|------|----------|------|
| 封面图 | 3:4 | 1242×1660 px | P1 封面 |
| 内页图 | 1:1 | 1080×1080 px | P2-P8 内页 |
| 结束页 | 3:4 | 1242×1660 px | P9 关注引导 |

### Keynote 设置

1. **打开 Keynote** → 新建空白文档
2. **文件** → 幻灯片大小 → 自定幻灯片大小
3. **输入尺寸**:
   - 封面：宽 1242 pt, 高 1660 pt
   - 内页：宽 1080 pt, 高 1080 pt

## 🎨 模板 1: 封面图模板

### 设计元素

```
┌─────────────────────────┐
│                         │
│    [AI/机器人图标]       │
│                         │
│   5 分钟搭建 AI 助手      │
│   (主标题 -72pt-白色)    │
│                         │
│   零基础 | 免费 | 开源   │
│   (副标题 -32pt-浅蓝)    │
│                         │
│   [打工人效率神器]       │
│   (角标 -24pt-黄色)      │
│                         │
│    [代码装饰元素]        │
│                         │
└─────────────────────────┘
```

### 配色方案

| 元素 | 颜色 | HEX | RGB |
|------|------|-----|-----|
| 背景渐变上 | 深蓝 | #1E3A8A | 30,58,138 |
| 背景渐变下 | 亮蓝 | #3B82F6 | 59,130,246 |
| 主标题 | 白色 | #FFFFFF | 255,255,255 |
| 副标题 | 浅蓝 | #93C5FD | 147,197,253 |
| 角标 | 亮黄 | #FCD34D | 252,211,77 |
| 装饰 | 青色 | #06B6D4 | 6,182,212 |

### 字体选择

- **主标题**: PingFang SC Bold (72pt)
- **副标题**: PingFang SC Regular (32pt)
- **角标**: PingFang SC Medium (24pt)

### 制作步骤

1. **设置背景**
   - 渐变填充 → 线性渐变
   - 上方颜色：#1E3A8A
   - 下方颜色：#3B82F6
   - 角度：90° (从上到下)

2. **添加主标题**
   - 文字："5 分钟搭建 AI 助手"
   - 字体：PingFang SC Bold, 72pt
   - 颜色：白色
   - 位置：居中偏上
   - 效果：添加轻微阴影 (不透明度 30%)

3. **添加副标题**
   - 文字："零基础 | 免费 | 开源"
   - 字体：PingFang SC Regular, 32pt
   - 颜色：#93C5FD
   - 位置：主标题下方 50pt

4. **添加角标**
   - 圆角矩形背景 (黄色 #FCD34D)
   - 文字："打工人效率神器"
   - 字体：PingFang SC Medium, 24pt
   - 颜色：深色 #1E3A8A
   - 位置：右上角

5. **添加装饰**
   - 代码符号：`{ } < > / *`
   - 线条装饰：细线 (1pt, 青色)
   - 位置：底部和四角

6. **导出**
   - 文件 → 导出到 → 图像
   - 格式：PNG
   - 质量：最高
   - 文件名：`P1-cover.png`

## 💡 设计技巧

### 提升质感的小技巧

1. **阴影效果**: 文字添加轻微阴影 (不透明度 20-30%)
2. **渐变背景**: 比纯色更有层次
3. **留白**: 不要填满，留出呼吸空间
4. **对齐**: 所有元素严格对齐
5. **一致性**: 所有图片用同一套配色

### 避免的坑

- ❌ 文字太小 (手机上看不到)
- ❌ 颜色太多 (不超过 3 种主色)
- ❌ 信息过载 (一张图一个重点)
- ❌ 模糊导出 (一定要 PNG 最高质量)
