## 🎮 交互功能

### 手势控制

| 手势 | 功能 | 说明 |
|------|------|------|
| **单指拖动** | 旋转视角 | 360° 环绕观察 |
| **双指拖动** | 平移视角 | 上下左右移动 |
| **捏合** | 缩放 | 拉近/拉远太阳 |
| **双击** | 重置 | 恢复默认视角 |

### 时间滑块

拖动底部时间滑块，可以：
- ⏪ **回溯时间** - 查看过去几小时的太阳位置
- ⏩ **预览未来** - 查看未来几小时的太阳位置
- 🌅 **看日出** - 观察太阳从东方升起
- 🌇 **看日落** - 观察太阳在西方落下

### 控制开关

- **显示轨迹** ✓/✗
  - 显示/隐藏太阳轨迹虚线
  
- **自动旋转** ✓/✗
  - 场景自动旋转展示全貌

## 🎨 视觉设计

### 3D 元素

1. **太阳** 🟡
   - 黄色发光球体
   - 橙色光晕效果
   - 半径：0.3 单位

2. **天空背景** 🌌
   - 顶部：深蓝色
   - 中部：中蓝色
   - 下部：浅蓝色
   - 地平线：淡蓝色

3. **地平线** ⬜
   - 反射平面
   - 反射率：30%

4. **方向标记** 🧭
   - N（北）- 白色文字
   - S（南）- 白色文字
   - E（东）- 白色文字

5. **太阳轨迹** 〰️
   - 橙色虚线
   - 从日出到日落的路径
   - 半透明效果

### 核心技术

- **SceneKit** - Apple 3D 引擎
- **SwiftUI** - 声明式 UI
- **Core Location** - 精准定位
- **天文算法** - NOAA 太阳位置计算

### 性能指标

- **帧率**: 60 FPS
- **多边形数**: < 1000
- **更新频率**: 每秒 1 次
- **内存占用**: < 50 MB

## 📖 教育价值

### 直观学习天文知识

通过 3D 可视化，可以轻松理解：

1. **太阳高度角** 
   - 夏天太阳高（仰角大）
   - 冬天太阳低（仰角小）

2. **方位角变化**
   - 早晨：东方（~90°）
   - 正午：南方（~180°）
   - 傍晚：西方（~270°）

3. **季节差异**
   - 夏至：轨迹高，白天长
   - 冬至：轨迹低，白天短

4. **纬度影响**
   - 赤道：太阳直射
   - 高纬：太阳斜射

## 🚀 使用方法

### 步骤 1: 打开 App

在 Xcode 中运行或安装到设备

### 步骤 2: 切换到 3D Tab

点击底部的 **"3D"** 标签（立方体图标）

### 步骤 3: 观察太阳

- 橙色球体 = 当前太阳位置
- 虚线 = 太阳轨迹
- 字母 = 方向标记

### 步骤 4: 交互操作

- 拖动旋转视角
- 捏合缩放
- 拖动时间滑块

### 步骤 5: 查看数据

- 右上角：实时数据
- 底部：详细数据卡片
