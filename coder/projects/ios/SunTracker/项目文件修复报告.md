## 📋 Xcode 项目文件结构

### .pbxproj 文件的关键段

1. **PBXBuildFile** - 声明哪些文件参与构建
2. **PBXFileReference** - 文件的元数据（类型、路径等）
3. **PBXGroup** - 文件在 Xcode 中的分组显示
4. **PBXSourcesBuildPhase** - 编译阶段包含的源文件

### 添加新文件的完整流程

```
1. 在 PBXBuildFile 添加构建引用
   ↓
2. 在 PBXFileReference 添加文件引用
   ↓
3. 在对应 PBXGroup 添加子项
   ↓
4. 在 PBXSourcesBuildPhase 添加编译项
```

## 💡 经验教训

### 为什么会出现这个问题？

**手动创建文件**时：
- ✅ 文件内容正确
- ❌ 但 Xcode 项目文件不会自动更新

**通过 Xcode 创建文件**时：
- ✅ Xcode 自动更新 .pbxproj 文件

### 最佳实践

1. **优先使用 Xcode 创建文件**
   - File → New → File...
   - Xcode 会自动处理项目文件

2. **手动创建文件后**
   - 在 Xcode 中右键项目
   - 选择 "Add Files to..."
   - 或手动编辑 .pbxproj

3. **使用命令行工具**
   - `xcodeproj` gem
   - 自动管理项目文件
