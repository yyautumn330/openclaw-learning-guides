# 🌅 3D 太阳可视化功能

## 功能介绍

全新的 **3D 太阳轨迹可视化** 界面，让你直观地看到太阳在天空中的位置！

### 特色功能

- 🎯 **实时太阳位置** - 橙色发光球体显示当前太阳位置
- 🌈 **动态天空背景** - 渐变蓝色模拟真实天空
- 📍 **方向标记** - N/S/E/W 四个方向指示
- 🌄 **太阳轨迹** - 虚线显示太阳全天的运动路径
- 🕐 **时间滑块** - 拖动查看不同时刻的太阳位置
- 📊 **数据面板** - 实时显示太阳仰角、方位角等数据

## 技术实现

### SceneKit 3D 引擎

使用 Apple 的 **SceneKit** 框架创建 3D 场景：

```swift
struct SceneKitView: UIViewRepresentable {
    func makeUIView(context: Context) -> SCNView {
        let scnView = SCNView()
        scnView.scene = createScene()
        scnView.allowsCameraControl = true  // 允许用户控制相机
        return scnView
    }
}
```

### 场景元素

1. **太阳** - 发光球体
   ```swift
   let sphere = SCNSphere(radius: 0.3)
   material.emission.contents = UIColor.orange  // 发光效果
   ```

2. **天空背景** - 渐变颜色
   ```swift
   LinearGradient(
       colors: [.深蓝, .中蓝，.浅蓝，.地平线]
   )
   ```

3. **地平线** - 反射平面
   ```swift
   let ground = SCNFloor()
   ground.reflectivity = 0.3
   ```

4. **方向标记** - N/S/E/W 文字
   ```swift
   let textGeometry = SCNText(string: "N", extrusionDepth: 0.1)
   ```

### 太阳位置计算

根据天文算法实时计算太阳的 3D 坐标：

```swift
// 将球坐标（仰角 + 方位角）转换为笛卡尔坐标
let x = distance * cos(elevation) * sin(azimuth)
let y = distance * sin(elevation)
let z = distance * cos(elevation) * cos(azimuth)
```

## 使用方法

### 1. 打开 3D 视图

在 App 底部点击 **"3D"** Tab

### 2. 查看当前太阳位置

- **橙色发光球体** = 太阳
- **高度** = 太阳仰角
- **水平位置** = 太阳方位角

### 3. 交互操作

| 手势 | 功能 |
|------|------|
| 单指拖动 | 旋转视角 |
| 双指拖动 | 平移视角 |
| 捏合 | 缩放 |
| 双击 | 重置视角 |

### 4. 时间滑块

拖动底部时间滑块，查看：
- 日出时太阳从东方升起
- 正午太阳到达最高点
- 日落时太阳在西方落下

### 5. 控制选项

- **显示轨迹** - 切换是否显示太阳路径虚线
- **自动旋转** - 场景自动旋转展示

## 数据面板

### 实时数据

显示在屏幕右上角：
- **仰角** - 太阳相对于地平线的高度（0-90°）
- **方位角** - 太阳的方向（0-360°，北=0°，东=90°）
- **时间** - 当前时间

### 底部数据卡片

- **仰角** - 精确到 0.1°
- **方位角** - 精确到 0.1°
- **可见性** - ☀️ 可见 / 🌙 不可见
- **方向** - 文字描述（北/东北/东等）

### 日出日落卡片

- **日出时间** - 黄色渐变卡片
- **日落时间** - 橙色渐变卡片
- **白天时长** - 自动计算

## 学习价值

通过 3D 可视化，你可以直观理解：

1. **太阳高度角** - 为什么夏天太阳高，冬天太阳低
2. **方位角变化** - 太阳从东到西的运动轨迹
3. **季节差异** - 不同日期太阳轨迹的不同
4. **纬度影响** - 不同地点看到的太阳位置差异
