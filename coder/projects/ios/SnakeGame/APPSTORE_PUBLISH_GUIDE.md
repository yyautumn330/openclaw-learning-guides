# 🚀 贪吃蛇 - App Store 发布完整指南

## 📋 发布前准备

### 1. Apple Developer 账号

**需要：** Apple Developer Program  membership（$99/年）

**注册流程：**
1. 访问 [Apple Developer](https://developer.apple.com)
2. 点击 "Account" → "Enroll"
3. 选择账号类型：
   - **个人/独资企业** - 最快（推荐）
   - **公司/组织** - 需要邓白氏编码
4. 填写信息并支付 $99/年
5. 等待审核（通常 24-48 小时）

**已有账号？** 跳过此步骤

---

### 2. 在 Xcode 中配置签名

#### 2.1 打开项目
```
打开 ~/Projects/SnakeGame/SnakeGame.xcodeproj
```

#### 2.2 配置签名
1. 点击左侧 **SnakeGame** 项目（蓝色图标）
2. 选择 **TARGETS** → **SnakeGame**
3. 点击 **Signing & Capabilities** 标签

#### 2.3 选择 Team
- 如果有账号：选择你的 Team
- 如果没有：点击 "Add an Account..." 登录 Apple ID

#### 2.4 配置 Bundle ID
```
Bundle Identifier: com.yourname.snakegame
```
**重要：** 改为唯一的标识符（替换 yourname）

#### 2.5 配置版本信息
在 **General** 标签：
- **Display Name:** 贪吃蛇
- **Version:** 1.0 (1)
- **Deployment Target:** iOS 15.0
- **Device Orientation:** 只选 Portrait（竖屏）

### 3. 准备应用素材

#### 3.1 应用图标（必须）

**要求：**
- 尺寸：1024 x 1024 像素
- 格式：PNG（无透明通道）
- 内容：不能有圆角（Apple 会自动添加）

**如何创建：**
1. 使用 [App Icon Generator](https://appicon.co)
2. 或使用 Xcode 的 Asset Catalog
3. 或找设计师设计

**放置位置：**
```
SnakeGame/Assets.xcassets/AppIcon.appiconset/
```

| 6.5" (iPhone 11 Pro Max) | 1242 x 2688 | 必需 |

### 4. 在 App Store Connect 创建应用

#### 4.1 登录 App Store Connect
访问：https://appstoreconnect.apple.com

#### 4.2 创建新 App
1. 点击 **"我的 App"**
2. 点击 **"+"** → **"创建新 App"**
3. 填写信息：

| 字段 | 填写内容 |
|------|---------|
| 平台 | iOS |
| 名称 | 贪吃蛇 |
| 主要语言 | 简体中文 |
| Bundle ID | com.yourname.snakegame（与 Xcode 一致）|
| SKU | snake-game-001（唯一标识） |
| 用户访问权限 | 完全访问 |

4. 点击 **"创建"**

---

### 5. 填写应用信息

#### 5.1 版本信息
- **版本：** 1.0
- **版权：** © 2026 Your Name

#### 5.2 价格与销售范围
1. 点击 **"价格与销售范围"**
2. **价格等级：** 选择 **等级 1（1 元）**
3. **销售范围：** 选择所有国家/地区
4. 点击 **"存储"**

#### 5.3 App 信息
填写以下内容：

**副标题：**
```
经典休闲游戏
```

**描述：** （使用上面准备的文案）

**关键词：**
```
贪吃蛇，休闲，游戏，经典，益智，单机，怀旧，街机
```

**技术支持 URL：**
```
https://yourdomain.com/support
```

**营销 URL（可选）：**
```
https://yourdomain.com/snakegame
```

**隐私政策 URL：**
```
https://yourdomain.com/privacy
```

#### 5.4 年龄分级
1. 点击 **"App 隐私"** → **"年龄分级"**
2. 选择 **4+**（适合 4 岁及以上）
3. 完成问卷（都选"无"或"很少"）

#### 5.5 App 隐私
1. 点击 **"App 隐私"**
2. 点击 **"开始"**
3. 选择 **"否，我们不会收集任何数据"**
4. 提交

---

### 6. 上传构建版本

#### 6.1 在 Xcode 中 Archive

1. **选择真机配置：**
   - 设备选择器选择 **"Any iOS Device (arm64)"**

2. **清理构建：**
   ```
   Product → Clean Build Folder
   (Shift + Cmd + K)
   ```

3. **Archive：**
   ```
   Product → Archive
   ```

4. **等待归档完成**（进度条在 Xcode 顶部）

#### 6.2 上传到 App Store Connect

1. **Organizer 窗口自动打开**
2. 选择刚才的 Archive
3. 点击 **"Distribute App"**
4. 选择 **"App Store Connect"**
5. 选择 **"Upload"**
6. 点击 **"Next"**
7. 保持默认选项，继续点击 **"Next"**
8. 点击 **"Upload"**
9. 等待上传完成

#### 6.3 验证上传

1. 访问 App Store Connect
2. 进入你的应用
3. 点击 **"1.0 准备提交"**
4. 在 **"构建版本"** 部分应该看到刚才上传的版本
5. 点击 **"+"** 选择构建版本

---

### 7. 提交审核

#### 7.1 完成最后检查

**提交前检查清单：**
- [ ] 应用图标已上传（1024x1024）
- [ ] 截图已上传（至少 3 张）
- [ ] 描述文案已填写
- [ ] 关键词已填写
- [ ] 价格设置为 1 元
- [ ] 隐私政策 URL 有效
- [ ] 构建版本已选择
- [ ] 年龄分级已完成

#### 7.2 回答审核问题

**出口合规：**
- 选择 **"否"**（游戏不涉及加密）

**广告标识符：**
- 选择 **"否"**（游戏没有广告）

#### 7.3 添加审核备注（可选）

如果有特殊情况需要说明：
```
无特殊说明。这是一个经典的贪吃蛇游戏。
```

#### 7.4 提交审核

1. 点击右上角 **"添加供审核的附件"**（如有需要）
2. 点击 **"提交以供审核"**
3. 确认提交

---

### 8. 等待审核

#### 8.1 审核时间
- **通常：** 24-48 小时
- **最长：** 3-5 个工作日

#### 8.2 查看状态
在 App Store Connect：
- **"我的 App"** → 选择应用 → **"App 审核"**

**可能的状态：**
- ⏳ 等待审核
- 🔍 正在审核
- ✅ 已批准
- ❌ 被拒绝（会说明原因）

#### 8.3 如果被拒绝

**常见拒绝原因：**

**4.3 重复应用**
- 解决：确保应用有独特性

**2.1 应用完整性**
- 解决：确保无崩溃，功能完整

**5.1.1 隐私**
- 解决：提供隐私政策

**收到拒绝通知后：**
1. 阅读拒绝原因
2. 修复问题
3. 重新提交审核

---

### 9. 发布应用

#### 9.1 审核通过后

**手动发布：**
1. 收到邮件通知
2. 登录 App Store Connect
3. 点击 **"发布"**

**自动发布：**
- 设置发布日期，到期自动上架

#### 9.2 应用上架

**上架后：**
- 可在 App Store 搜索"贪吃蛇"找到
- 分享链接：https://apps.apple.com/app/idXXXXXXXXX
