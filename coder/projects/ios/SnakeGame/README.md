# 🐍 贪吃蛇 - iOS 游戏

经典的贪吃蛇游戏，使用 Swift 和 SwiftUI 开发。

## 📱 系统要求

- iOS 15.0+
- Xcode 15.0+
- Swift 5.9+

## 🎮 游戏特性

- ✅ 经典贪吃蛇玩法
- ✅ 简洁现代的 UI 设计
- ✅ 流畅的动画效果
- ✅ 最高分记录
- ✅ 暂停/继续功能
- ✅ 难度递增（速度越来越快）

## 🛠️ 构建说明

### 1. 打开项目

```bash
cd ~/Projects/SnakeGame
open SnakeGame.xcodeproj
```

### 2. 配置签名

在 Xcode 中：
1. 选择项目 → Signing & Capabilities
2. 选择你的 Team（需要 Apple Developer 账号）
3. Bundle Identifier 设置为唯一值，如：`com.yourname.snakegame`

### 3. 构建运行

- 模拟器：选择模拟器，按 `Cmd + R`
- 真机：连接 iPhone，选择设备，按 `Cmd + R`

## 🚀 发布到 App Store

### 步骤 1：创建 App Store Connect 应用

1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 点击"我的 App" → "+" → "创建新 App"
3. 填写信息：
   - **名称**：贪吃蛇
   - **主要语言**：简体中文
   - **Bundle ID**：与 Xcode 中一致
   - **SKU**：snake-game-001
   - **用户访问权限**：完全访问

### 步骤 2：配置应用信息

在 App Store Connect 中填写：

#### 价格与销售范围
- **价格等级**：1 元（等级 1）
- **销售范围**：选择所有国家/地区

#### 应用信息
- **副标题**：经典休闲游戏
- **描述**：
  ```
  经典贪吃蛇游戏回来了！

  游戏特色：
  - 简洁易用的操作
  - 流畅的游戏体验
  - 最高分记录
  - 适合所有年龄段

  控制方式：
  - 点击方向按钮控制蛇的移动
  - 吃到红色食物可以得分并增长
  - 避免撞墙和撞到自己

  快来下载挑战你的最高分吧！
  ```
- **关键词**：贪吃蛇，休闲，游戏，经典，益智
- **支持 URL**：你的网站（可选）
- **营销 URL**：你的网站（可选）

#### 隐私政策 URL
需要提供一个隐私政策链接，可以使用：
- 自己的网站
- GitHub Pages
- 隐私政策生成工具

### 步骤 3：准备截图

需要准备以下尺寸的截图（单位：像素）：

| 设备 | 尺寸 |
|------|------|
| 6.7" (iPhone 14 Pro Max) | 1290 x 2796 |
| 6.5" (iPhone 11 Pro Max) | 1242 x 2688 |
| 5.5" (iPhone 8 Plus) | 1242 x 2208 |

### 步骤 4：上传构建版本

在 Xcode 中：
1. 选择 "Any iOS Device (arm64)"
2. 菜单：Product → Archive
3. 等待归档完成
4. 在 Organizer 中点击 "Distribute App"
5. 选择 "App Store Connect"
6. 选择 "Upload"
7. 跟随向导完成上传

### 步骤 5：提交审核

在 App Store Connect 中：
1. 选择刚才上传的构建版本
2. 填写版本信息（版本号 1.0）
3. 回答出口合规问题（通常选 No）
4. 点击"添加供审核的附件"（如有需要）
5. 点击"提交以供审核"

### 步骤 6：等待审核

- 审核时间：通常 24-48 小时
- 可以在"App 审核"页面查看状态
- 如有问题，会收到邮件通知

### 步骤 7：发布

审核通过后：
- 手动发布：在 App Store Connect 点击"发布"
- 自动发布：设置自动发布日期
