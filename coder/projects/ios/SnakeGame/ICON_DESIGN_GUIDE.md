# 🎨 贪吃蛇 - 应用图标设计方案

## 📐 图标规格

### Apple 官方要求

| 要求 | 说明 |
|------|------|
| 尺寸 | 1024 x 1024 像素 |
| 格式 | PNG（无透明通道） |
| 圆角 | ❌ 不要添加（Apple 会自动添加） |
| 文字 | ❌ 避免使用文字 |
| 背景 | 可以是任何颜色 |

## 🛠️ 如何创建图标

### 方法 1：使用 Canva（免费，推荐）

**步骤：**

1. **访问 Canva**
   ```
   https://www.canva.com
   ```

2. **创建设计**
   - 点击"创建设计"
   - 选择"自定义尺寸"
   - 输入：1024 x 1024

3. **设计图标**
   - 添加圆形（蛇头）
   - 添加眼睛（小圆形）
   - 添加舌头（三角形）
   - 设置背景色

4. **下载**
   - 点击"下载"
   - 选择 PNG 格式
   - 确保无透明背景

**Canva 模板搜索关键词：**
- "App Icon"
- "Snake Logo"
- "Game Icon"

---

### 方法 2：使用 Figma（免费，专业）

**步骤：**

1. **访问 Figma**
   ```
   https://www.figma.com
   ```

2. **创建文件**
   - New Design File
   - 画布尺寸：1024 x 1024

3. **绘制图标**
   - 使用 Shape 工具绘制蛇头
   - 添加细节（眼睛、舌头）
   - 设置颜色和阴影

4. **导出**
   - 选择图形
   - 右键 → Export
   - 选择 PNG，1x

**Figma 优势：**
- 更专业的设计工具
- 可以添加渐变和阴影
- 易于调整和优化

## 📱 测试图标

### 在设备上预览

**方法 1：发送到手机**
1. 将图标发送到 iPhone
2. 在相册中全屏查看
3. 检查清晰度和色彩

**方法 2：模拟器测试**
1. 在 Xcode 中运行项目
2. 替换 Assets.xcassets/AppIcon
3. 查看模拟器效果

**方法 3：在线预览**
- 使用 [App Icon Preview](https://www.appicon.co)
- 上传图标查看在不同设备上的效果

## 💡 设计技巧

### DO（推荐）
✅ 使用简洁的图形
✅ 高对比度配色
✅ 突出主要元素
✅ 保持风格一致
✅ 测试缩小效果

### DON'T（避免）
❌ 添加文字（太小看不清）
❌ 使用过多细节
❌ 颜色太复杂
❌ 添加圆角（Apple 会自动加）
❌ 使用透明背景
