## 🎯 应用图标设计

### 设计要求

**Apple 官方要求：**
- 尺寸：1024 x 1024 像素
- 格式：PNG（无透明通道）
- 内容：不能有圆角（Apple 会自动添加）
- 风格：简洁、易识别

### 设计方案

**方案 A：蛇头图标**
```
🟩 绿色蛇头（带眼睛）
⬛ 黑色背景
🔴 红色舌头
```

**方案 B：蛇 + 苹果**
```
🟩 绿色蛇缠绕
🔴 红色苹果
⬜ 白色/浅色背景
```

**方案 C：像素风格**
```
🟩 像素风蛇头
⬛ 复古游戏风格
```

### 如何创建图标

#### 方法 1：使用在线工具（推荐）

1. **Canva**（免费）
   - 访问：https://www.canva.com
   - 搜索 "App Icon"

   - 访问：https://www.figma.com

## 📄 隐私政策模板

### 简单版（推荐）

已为你准备好隐私政策 HTML 文件：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>隐私政策 - 贪吃蛇</title>
    <style>
        body { font-family: -apple-system, sans-serif; padding: 20px; }
        h1 { color: #333; }
        h2 { color: #666; }
    </style>
</head>
<body>
    <h1>隐私政策</h1>
    <p>最后更新：2026 年 2 月 27 日</p>
    
    <h2>1. 信息收集</h2>
    <p>本应用（贪吃蛇）不收集任何个人信息。</p>
    
    <h2>2. 数据使用</h2>
    <p>游戏仅在本地存储最高分记录，不会传输任何数据到服务器或第三方。</p>
    
    <h2>3. 第三方服务</h2>
    <p>本应用不使用任何第三方服务、分析工具或广告 SDK。</p>
    
    <h2>4. 数据安全</h2>
    <p>所有数据仅存储在您的设备本地，我们不会访问或收集这些数据。</p>
    
    <h2>5. 儿童隐私</h2>
    <p>本应用适合所有年龄段，不针对儿童收集任何信息。</p>
    
    <h2>6. 隐私政策变更</h2>
    <p>我们可能会不时更新本隐私政策。变更将在本页面公布。</p>
    
    <h2>7. 联系我们</h2>
    <p>如有任何问题，请联系：support@yourdomain.com</p>
</body>
</html>
```

### 如何托管隐私政策

#### 方法 1：GitHub Pages（免费，推荐）

1. **创建 GitHub 账号**
   - 访问：https://github.com

2. **创建仓库**
   ```
   仓库名：snakegame-privacy
   可见性：Public
   ```

3. **上传 privacy.html**
   - 将上面的 HTML 保存为 `index.html`
   - 上传到仓库

4. **启用 GitHub Pages**
   - Settings → Pages
   - Source: main branch
   - 获得 URL：https://yourname.github.io/snakegame-privacy/

5. **使用 URL**
   ```
   https://yourname.github.io/snakegame-privacy/
   ```

#### 方法 2：自己的网站

如果有网站，直接上传 HTML 文件即可。
