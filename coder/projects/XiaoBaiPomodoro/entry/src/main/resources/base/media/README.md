# 媒体资源说明

## 图标文件

请在此目录下放置以下图标文件：

### app_icon.png
- 应用图标
- 尺寸：512x512 像素
- 格式：PNG
- 建议：使用番茄图标设计

### icon.png
- 启动图标
- 尺寸：512x512 像素
- 格式：PNG
- 建议：与 app_icon 保持一致

## 图标设计建议

小白番茄专注钟的图标设计建议：
- 主色调：红色 (#FF6B6B)
- 元素：番茄图案
- 风格：简洁、现代
- 背景：透明或白色

## 临时方案

开发测试期间，可以使用系统默认图标：
1. 复制任意 512x512 PNG 图片到此目录
2. 重命名为 app_icon.png 和 icon.png
3. 重新构建项目
