## 📋 使用建议

### 应用图标使用
1. **iOS App Store**: 使用 `app-icon-512.png`
2. **需要其他尺寸**: 从 512x512 缩放生成
   - 1024x1024 (App Store 展示)
   - 180x180 (iPhone @3x)
   - 120x120 (iPhone @2x)

### 应用截图使用
1. **App Store**: 使用 1080x1920 PNG 文件
2. **宣传材料**: 可直接使用或添加设备外壳
3. **需要修改**: 编辑对应的 HTML 源文件

### 设计稿预览
- `ui-design-preview.jpg` 可用于快速预览和分享
- HTML 源文件可在浏览器中查看交互效果

1. **统一视觉语言**: 番茄红贯穿所有页面

4. **游戏化元素**: 成就系统增加用户粘性
