# 小白番茄专注钟 - 应用图标设计说明

## 📋 设计概述

本图标采用分层设计理念，使用程序化方式生成简洁、专业的番茄钟应用图标。

---

## 🎨 设计要点

### 分层结构

1. **背景层** - 纯白色圆形背景 (#FFFFFF)
2. **主体层** - 番茄主体（红色圆形 #FF6B6B）
3. **装饰层** - 三片绿叶点缀 (#4CAF50)
4. **效果层** - 高光和阴影增强立体感

### 颜色规范

| 元素 | 颜色代码 | 说明 |
|------|----------|------|
| 番茄红 | `#FF6B6B` | 主色调，温暖活泼 |
| 叶子绿 | `#4CAF50` | 自然清新 |
| 背景白 | `#FFFFFF` | 简洁干净 |
| 高光 | `rgba(255,255,255,0.4)` | 增强立体感 |
| 阴影 | `rgba(0,0,0,0.2)` | 增加层次 |

---

## 🛠️ 技术规格

### 文件格式
- **格式**: PNG（支持透明背景）
- **尺寸**: 512x512 像素
- **色彩模式**: RGBA
- **文件大小**: ~14KB（优化后）

### 交付文件

| 文件名 | 用途 | 尺寸 |
|--------|------|------|
| `app-icon-512.png` | 主图标（透明背景） | 512x512 |
| `app-icon-512-square.png` | 正方形版本（带圆角） | 512x512 |

## 💻 生成方式

### 方案 A: Python + Pillow（本次使用）

使用 Python PIL/Pillow 库程序化生成图标，优势：
- ✅ 无需外部 API 密钥
- ✅ 完全可控的分层设计
- ✅ 可重复生成
- ✅ 快速迭代

**生成脚本**: `generate_icon.py`

```bash
# 运行生成
python3 generate_icon.py
```

### 方案 B: ComfyUI 工作流（备选）

如需使用 AI 生成，推荐以下 ComfyUI 工作流：

#### 工作流节点配置

1. **Load Checkpoint**
   - 模型：SDXL 或 FLUX.1
   - 推荐：`sd_xl_base_1.0.safetensors`

2. **CLIP Text Encode (Positive Prompt)**
   ```
   minimalist tomato app icon, flat design, simple red tomato 
   with green leaves, white background, professional app store 
   icon, clean vector style, 512x512
   ```

3. **CLIP Text Encode (Negative Prompt)**
   ```
   text, watermark, blurry, low quality, distorted, ugly, 
   deformed, noisy, cluttered, complex background
   ```

4. **KSampler**
   - Steps: 20-30
   - CFG: 7.0
   - Sampler: euler or dpmpp_2m

5. **Upscale**
   - 目标尺寸：512x512
   - 算法：Lanczos 或 Bicubic

6. **Save Image**
   - 格式：PNG
   - 质量：最高

#### 提示词优化

**基础提示词**:
```
minimalist tomato app icon, flat design, simple red tomato 
with green leaves, white background, professional app store 
icon, clean vector style
```

**增强提示词**（更详细）:
```
professional app icon, cute red tomato character, flat vector 
illustration, minimal design, two green leaves on top, 
white circular background, iOS app store style, clean lines, 
vibrant colors, #FF6B6B tomato red, #4CAF50 leaf green, 
high contrast, centered composition, 512x512
```

**负面提示词**:
```
text, words, letters, watermark, signature, blurry, 
low resolution, distorted, deformed, ugly, noisy, 
cluttered, complex background, photorealistic, 3d render
```

## 📐 设计原则

### 1. 简洁性
- 去除多余细节
- 保持图形清晰易识别
- 适合小尺寸显示

### 2. 一致性
- 与应用 UI 风格统一
- 颜色符合品牌规范
- 视觉语言连贯

### 3. 可识别性
- 番茄形状清晰
- 颜色对比度高
- 在不同背景下都可见

### 4. 专业性
- 符合应用商店规范
- 边缘平滑无锯齿
- 色彩准确

## 🔄 迭代建议

### 如需调整

1. **颜色调整**
   - 修改 `generate_icon.py` 中的颜色常量
   - 重新运行脚本

2. **形状调整**
   - 调整 `tomato_radius` 参数
   - 修改叶子位置和大小

3. **添加元素**
   - 可在番茄上添加简单表情
   - 或添加小白（角色）元素
