## 技术实现细节

### 数据存储
- 使用 `@ohos.data.preferences` 进行本地持久化
- 数据结构：JSON 序列化
- 自动保存时机：番茄完成、设置变更、成就解锁

### 成就检测逻辑
- 实时检测：每次番茄完成后检查所有成就条件
- 时间相关成就：基于完成时的系统时间判断
- 连续天数：基于上次使用日期计算

### 统计图表
- 使用 Column/Row 组件自绘柱状图
- 动态计算柱高 (基于最大值归一化)
- 颜色区分：今天 (主色) vs 其他 (辅助色)

### UI 设计
- 卡片式布局，圆角 12px
- 轻微阴影效果
- 主色：#FF6B6B (番茄红)
- 辅助色：#4ECDC4 (青绿)
- 背景色：#F7F7F7 (浅灰)

## 使用说明

### 设置功能
1. 点击底部导航栏「设置」标签
2. 使用滑块调整各项时长
3. 点击「保存设置」按钮
4. 或点击「重置默认」恢复初始值

### 成就系统
1. 点击底部导航栏「成就」标签
2. 查看成就进度和解锁状态
3. 完成番茄自动解锁成就
4. 已解锁成就显示解锁日期

### 统计功能
1. 点击底部导航栏「统计」标签
2. 查看今日/总计数据
3. 切换周/月视图查看趋势
4. 查看最佳记录
