# 自我提升学习应用 - 整体计划书

**制定日期**: 2026-03-10  
**项目负责人**: 小白 (CM-Dev)  
**项目代号**: GrowthHub (成长中心)

---

## 📋 项目背景

### 来源
基于小白番茄专注钟开发过程中积累的技术和经验，打造一个更全面的个人成长管理应用。

### 核心理念
> "专注 + 学习 + 反思 = 持续成长"

---

## 🎯 产品定位

### 目标用户
- 希望提升自我的职场人士
- 需要时间管理的学生群体
- 追求持续成长的知识工作者

### 核心价值
| 价值维度 | 说明 |
|---------|------|
| **专注管理** | 番茄工作法 + 深度工作时段 |
| **学习追踪** | 技能树 + 学习进度管理 |
| **反思成长** | 每日/每周复盘 + 成长日记 |
| **成就激励** | 成长可视化 + 成就系统 |

---

## 📱 功能规划

### 核心功能模块

#### 1️⃣ 专注模块 (Focus)
**复用番茄钟核心功能**

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 番茄计时器 | 25 分钟专注 + 5 分钟休息 | P0 |
| 自定义时长 | 支持 5-120 分钟灵活设置 | P0 |
| 专注统计 | 日/周/月专注时长统计 | P0 |
| 专注标签 | 为不同任务打标签分类 | P1 |
| 白噪音 | 雨声、咖啡馆等背景音 | P2 |
| Forest 模式 | 专注种树，增加趣味性 | P2 |

#### 2️⃣ 学习模块 (Learn) ⭐ 核心差异化
**新增功能 - 技能树系统**

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 技能树 | 可视化技能成长路径 | P0 |
| 学习记录 | 记录每次学习内容和时长 | P0 |
| 进度追踪 | 显示技能完成百分比 | P0 |
| 学习资源 | 关联书籍/课程/文章链接 | P1 |
| 学习笔记 | 关联学习笔记和心得 | P1 |
| 学习提醒 | 每日学习目标提醒 | P2 |
| 技能卡片 | 分享学习成果到社交网络 | P2 |

**技能树示例**:
```
📚 鸿蒙开发 (65%)
├── ✅ ArkTS 基础 (100%)
├── ✅ ArkUI 组件 (100%)
├── 🔄 后台任务管理 (60%)
└── ⏳ 分布式能力 (20%)

🎨 UI 设计 (40%)
├── ✅ 设计原则 (100%)
├── 🔄 色彩理论 (50%)
└── ⏳ 排版布局 (20%)
```

#### 3️⃣ 反思模块 (Reflect) ⭐ 核心差异化
**新增功能 - 成长复盘系统**

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 每日复盘 | 3 问模板（完成/不足/改进） | P0 |
| 每周回顾 | 周度目标和成就总结 | P0 |
| 成长日记 | 记录重要时刻和感悟 | P1 |
| 情绪追踪 | 记录每天的心情状态 | P2 |
| 月度总结 | 月度成长和反思报告 | P2 |

**每日复盘模板**:
```
📅 2026-03-10 复盘

✅ 今日完成
- 完成番茄钟 8 个
- 学习鸿蒙开发 2 小时
- 完成 P1 改进

🤔 不足之处
- 上午效率较低
- 运动计划未完成

💡 明日改进
- 早起先做最重要的事
- 晚上安排 30 分钟运动
```

#### 4️⃣ 成就模块 (Achieve)
**升级番茄钟成就系统**

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 成就系统 | 50+ 成长相关成就 | P0 |
| 等级系统 | 经验值 + 等级成长 | P1 |
| 成长报告 | 周/月度成长数据报告 | P1 |
| 勋章墙 | 可视化成就展示 | P1 |
| 社交分享 | 分享成就到朋友圈 | P2 |

**成就示例**:
```
🏆 早起鸟儿 - 连续 7 天 6 点前起床
🏆 专注达人 - 单日完成 10 个番茄钟
🏆 学习狂人 - 周学习时长 20 小时
🏆 坚持就是胜利 - 连续 30 天打卡
🏆 全能选手 - 专注/学习/反思全部达标
```

#### 5️⃣ 个人中心 (Profile)

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 个人主页 | 展示成长数据和技能树 | P0 |
| 目标设定 | 设定周/月/年度目标 | P0 |
| 数据导出 | 导出学习数据和报告 | P1 |
| 主题设置 | 深色/浅色模式切换 | P0 |
| 提醒设置 | 自定义各类提醒 | P1 |

## 🏗️ 技术架构

### 技术栈
| 层级 | 技术选型 |
|------|---------|
| **前端框架** | HarmonyOS ArkTS + ArkUI |
| **状态管理** | 单例模式 + EventBus |
| **数据持久化** | Preferences + 本地数据库 |
| **设计系统** | DesignTokens (复用番茄钟) |
| **组件库** | Toast + 自定义组件 |

### 架构设计
```
┌─────────────────────────────────────────┐
│              UI Layer                    │
│  (Index, Focus, Learn, Reflect, Profile)│
├─────────────────────────────────────────┤
│           Component Layer                │
│    (Toast, SkillTree, Diary, Charts)    │
├─────────────────────────────────────────┤
│            Model Layer                   │
│  (UserModel, SkillModel, DiaryModel)    │
├─────────────────────────────────────────┤
│           Service Layer                  │
│  (TimerService, SyncService, Notify)    │
├─────────────────────────────────────────┤
│          Utility Layer                   │
│  (EventBus, DesignTokens, Utils)        │
└─────────────────────────────────────────┘
```
