## 🎯 功能详情

### 1. 节奏模式

#### 匀速模式（Constant）
- 固定 BPM 持续播放
- 适合稳定配速跑步

#### 渐进加速模式（Gradual Acceleration）
- 从起始 BPM 线性增加到目标 BPM
- 默认配置：160 BPM → 180 BPM（30 分钟）
- 适合长跑渐进加速

#### 间歇训练模式（Interval）
- 高 BPM 冲刺 → 低 BPM 恢复，循环
- 默认配置：180 BPM 冲刺 60 秒 → 160 BPM 恢复 120 秒（5 次循环）
- 适合 HIIT 训练

### 2. 配速 BPM 联动

**算法**：
```
BPM = (60 / pace) * 18
```

**示例**：
- 6:00/km → 180 BPM
- 5:00/km → 216 BPM（限制到 200）
- 8:00/km → 135 BPM（限制到 140）

**特性**：
- 平滑调整（避免突变）
- 灵敏度可调
- 自动限制范围（140-200 BPM）

### 3. 语音提示

**提示类型**：
- 节拍数提示（每 20 拍）
- BPM 变化提示（加速/减速）
- 节奏阶段提示（冲刺/恢复）
- 配速提醒

**语音内容**：
- "保持节奏，当前 180 步每分钟"
- "加速到 180 步每分钟"
- "开始冲刺 1/5"
- "当前配速 6 分 0 秒每公里"

### 4. 后台播放

**实现方式**：
- 前台服务（ContinuousTask）
- 通知栏显示（"节拍器运行中：180 BPM - 匀速保持"）
- 锁屏持续播放

**优势**：
- 系统不会杀死应用
- 用户可随时查看状态
- 支持蓝牙耳机
