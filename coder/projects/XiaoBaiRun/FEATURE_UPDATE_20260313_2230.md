## 🔧 技术实现

### Metronome.ets

**核心状态**:
```typescript
@State isPlaying: boolean = false;
@State bpm: number = 160;
@State volume: number = 0.7;
@State beatCount: number = 0;
```

**节拍计时器**:
```typescript
startBeatTimer() {
  const intervalMs = 60000 / this.bpm;
  this.beatTimer = setInterval(() => {
    this.beatCount++;
  }, intervalMs);
}
```
