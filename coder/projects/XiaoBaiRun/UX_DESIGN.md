## 📱 页面设计

### 1. 跑步首页 (Index)

**布局结构**:
```
┌─────────────────────────┐
│ 顶部标题栏              │
├─────────────────────────┤
│                         │
│   地图区域 (WebView)     │
│   + 轨迹绘制 (Canvas)    │
│                         │
├─────────────────────────┤
│ 实时数据展示            │
│ 时间 | 距离 | 配速       │
├─────────────────────────┤
│ 控制按钮                │
│ [开始] [暂停] [停止]     │
└─────────────────────────┘
```

**交互流程**:
1. 打开应用 → 请求定位权限
2. 点击"开始" → 开始记录轨迹
3. 实时显示时间/距离/配速
4. 点击"暂停" → 暂停记录
5. 点击"停止" → 结束跑步，保存记录

## 🎯 交互规范

### 按钮交互
- **点击态**: 缩放 0.95
- **禁用态**: 透明度 0.5
- **加载态**: 显示 Loading 动画

### 列表交互
- **下拉刷新**: 刷新数据
- **上拉加载**: 加载更多
- **左滑删除**: 删除记录

### 地图交互
- **双指缩放**: 缩放地图
- **单指拖动**: 移动地图
- **点击标记**: 查看详细信息

## 🎨 深色模式适配

### 自动切换
- 跟随系统深色模式设置
- 手动开关覆盖

### 适配原则
- 背景色：浅 → 深
- 文字色：深 → 浅
- 主色调：降低饱和度
- 轨迹颜色：保持一致性
