## 🔧 技术限制说明

### 地图功能
- **限制**: 使用静态图片，非交互式地图
- **原因**: 高德 SDK 需要 `useNormalizedOHMUrl` 配置 (当前 SDK 不支持)
- **影响**: 无法拖动地图，缩放需要重新加载图片
- **解决**: 等待 HarmonyOS SDK 更新或升级到支持版本

### 音频播放
- **限制**: 使用日志模拟，无真实声音
- **原因**: HarmonyOS 音频 API (`@ohos.multimedia.audio` / `@kit.MediaKit`) 类型与文档不符
- **影响**: 开发调试时可通过日志确认节拍，真机无声音
- **解决**: 等待 HarmonyOS SDK 更新或使用系统音效 API
