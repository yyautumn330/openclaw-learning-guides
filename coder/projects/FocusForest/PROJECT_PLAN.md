## 📋 项目概述

### 产品定位
专注森林 = 番茄钟 + 白噪音 + 种树游戏化

**核心理念**: "专注种树，收获森林"

### 目标用户
- 需要专注学习/工作的学生/职场人士
- 喜欢游戏化激励的用户
- 白噪音辅助专注需求者

### 差异化特点
| 特性 | 说明 |
|------|------|
| **游戏化激励** | 专注成功种树，失败树枯萎 |
| **白噪音** | 多种自然音效辅助专注 |
| **森林统计** | 可视化专注成果 |
| **社交专注** | 好友一起专注种树 (V2.0) |
