# 专注森林 (FocusForest) - 产品规格说明书

**版本**: 1.0.0  
**创建日期**: 2026-03-11  
**目标发布**: 2026-03-24

---

## 📱 产品概述

### 产品名称
**专注森林 (FocusForest)**

### 产品定位
游戏化专注力培养应用 - 通过种树游戏机制帮助用户培养专注习惯

### 目标用户
- 需要提升专注力的学生
- 需要深度工作的职场人士
- 希望培养良好习惯的用户

### 核心价值
- 🎮 **游戏化激励** - 专注成功种树，失败树枯萎
- 🌲 **可视化成长** - 森林成长记录专注历程
- 🎵 **白噪音辅助** - 6 种自然音效帮助专注
- 📊 **数据统计** - 专注时长、树木统计

---

## 🎯 功能规格

### V1.0 核心功能

#### 1. 专注计时器
| 功能 | 规格 | 优先级 |
|------|------|--------|
| 默认时长 | 25 分钟 | P0 |
| 自定义时长 | 5-120 分钟可调 | P0 |
| 暂停/继续 | 支持 | P0 |
| 重置功能 | 支持 | P0 |
| 后台运行 | 支持 | P0 |

#### 2. 种树系统
| 功能 | 规格 | 优先级 |
|------|------|--------|
| 树木类型 | 8 种 (幼苗/松树/橡树/樱花树/枫树/竹子/棕榈树/金色传说树) | P0 |
| 解锁机制 | 根据专注总时长解锁 | P0 |
| 树木成长 | 专注成功树木成长 | P0 |
| 树木枯萎 | 专注失败树木枯萎 | P0 |
| 森林展示 | 可视化展示所有树木 | P0 |

#### 3. 白噪音
| 功能 | 规格 | 优先级 |
|------|------|--------|
| 音效类型 | 6 种 (雨声/咖啡馆/海浪/森林/篝火/夜雨) | P0 |
| 音效选择 | 专注前可选择 | P0 |
| 音量控制 | 独立音量调节 | P1 |
| 音效混合 | 支持多种音效混合 | P2 |

#### 4. 数据统计
| 功能 | 规格 | 优先级 |
|------|------|--------|
| 今日统计 | 完成番茄数、专注时长 | P0 |
| 总计统计 | 总番茄数、总专注时长 | P0 |
| 周统计 | 近 7 天柱状图 | P0 |
| 月统计 | 近 30 天柱状图 | P0 |
| 最佳记录 | 单日最高记录 | P0 |
| 树木统计 | 各树种数量统计 | P0 |

#### 5. 成就系统
| 功能 | 规格 | 优先级 |
|------|------|--------|
| 成就数量 | 10 个基础成就 | P0 |
| 成就解锁 | 自动解锁 | P0 |
| 成就展示 | 成就墙展示 | P0 |
| 成就通知 | 解锁时 Toast 提示 | P0 |
