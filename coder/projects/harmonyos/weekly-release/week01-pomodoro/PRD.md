# 小白番茄专注钟 - 产品需求文档 (PRD)

**项目名称**: 小白番茄专注钟 (XiaoBai Pomodoro)  
**版本**: v1.0  
**类型**: 效率工具 - 鸿蒙原生应用  
**目标市场**: 鸿蒙应用市场  
**开发周期**: 7 天 (3.7-3.13)  
**上线日期**: 2026-03-13

---

## 1. 产品概述

### 1.1 产品定位
一款基于番茄工作法的专注计时工具，帮助用户提升工作和学习效率。

### 1.2 目标用户
- **主要用户**: 大学生、考研党、自由职业者
- **次要用户**: 办公室白领、需要专注工作的人群
- **用户痛点**: 
  - 容易分心，无法长时间专注
  - 缺乏时间管理意识
  - 需要数据反馈激励

### 1.3 核心价值
- 🍅 经典番茄工作法，25 分钟专注 +5 分钟休息
- 📊 数据统计，可视化专注时长
- 🏆 成就系统，激励用户坚持
- 📱 鸿蒙原生，流畅体验

---

## 2. 功能需求

### 2.1 核心功能 (P0)

| 功能 | 描述 | 优先级 | 工时 |
|------|------|--------|------|
| **番茄计时器** | 25 分钟倒计时，自动提醒休息 | P0 | 4h |
| **休息计时器** | 5 分钟短休息，可延长 | P0 | 2h |
| **专注记录** | 记录每次专注时长和完成数量 | P0 | 3h |
| **数据统计** | 日/周/月专注时长统计图表 | P0 | 4h |

### 2.2 增强功能 (P1)

| 功能 | 描述 | 优先级 | 工时 |
|------|------|--------|------|
| **自定义时长** | 可调整番茄钟和休息时长 | P1 | 2h |
| **白噪音** | 提供雨声、咖啡馆等背景音 | P1 | 3h |
| **成就系统** | 连续打卡、累计时长等成就 | P1 | 3h |
| **桌面小组件** | 鸿蒙桌面卡片，快速启动 | P1 | 4h |

### 2.3 扩展功能 (P2)

| 功能 | 描述 | 优先级 | 工时 |
|------|------|--------|------|
| **专注目标** | 设置每日/每周专注目标 | P2 | 2h |
| **分享功能** | 分享专注成果到社交平台 | P2 | 2h |
| **云同步** | 多设备数据同步 | P2 | 4h |

## 3. 界面设计

### 3.1 页面结构

```
┌─────────────────────────────────────┐
│           首页 (计时器)              │
├─────────────────────────────────────┤
│  - 大数字倒计时显示                  │
│  - 开始/暂停/重置按钮                │
│  - 当前番茄数显示                    │
│  - 底部导航：首页/统计/我的          │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│           统计页面                   │
├─────────────────────────────────────┤
│  - 日/周/月切换                      │
│  - 柱状图/折线图展示                 │
│  - 累计专注时长                      │
│  - 完成番茄数                        │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│           我的页面                   │
├─────────────────────────────────────┤
│  - 成就徽章展示                      │
│  - 设置入口                          │
│  - 关于我们                          │
└─────────────────────────────────────┘
```

### 3.2 设计风格
- **设计语言**: 鸿蒙设计规范 (HarmonyOS Design)
- **主色调**: 番茄红 (#FF6B6B) + 清新绿 (#4ECDC4)
- **字体**: 鸿蒙字体 (HarmonyOS Sans)
- **图标**: 扁平化 + 微渐变

### 3.3 交互要求
- 启动时间 < 1 秒
- 页面切换流畅无卡顿
- 支持深色模式
- 支持横屏显示

## 4. 技术实现

### 4.1 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| **开发语言** | ArkTS | 鸿蒙主推语言 |
| **UI 框架** | ArkUI | 声明式 UI |
| **数据存储** | Preferences + RDB | 轻量级存储 |
| **图表库** | 自研或使用开源 | 统计图表 |
| **推送通知** | Notification Kit | 休息提醒 |

### 4.2 鸿蒙特性利用
- ✅ 鸿蒙桌面卡片 (Service Widget)
- ✅ 鸿蒙通知系统
- ✅ 鸿蒙深色模式适配
- ✅ 鸿蒙分布式能力 (预留)

### 4.3 性能要求
- 冷启动 < 1 秒
- 内存占用 < 50MB
- 电量消耗 < 1%/小时
- 支持后台计时
