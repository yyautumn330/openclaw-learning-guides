## 🔧 管理命令

```bash
# 查看所有已安装技能
skills list -g

# 搜索新技能
skills find <关键词>

# 安装新技能
skills add <owner/repo@skill> -g -y

# 删除技能
skills remove <skill-name>

# 检查更新
skills check

# 更新所有技能
skills update
```
